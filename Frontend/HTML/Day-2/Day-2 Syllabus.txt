 Anchor tag (<a>)
< a href = "https://www.google.com" > Google < /a>

| Attribute         | Description                |
| ----------------- | -------------------------- |
| `href`            | URL of the page or file    |
| `target="_blank"` | Opens link in new tab      |
| `download`        | Downloads the file         |
| `mailto:`         | Opens default email client |
| `tel:`            | Opens phone dialer         |
| `id`              | Unique identifier          |
| `class`           | CSS class for styling

üî∏ Ordered List (<ol>)

An ordered list displays items in a numbered sequence ‚Äî showing priority, rank, or step-by-step instructions.


<ol>  
  <li>Item 1</li>
  <li>Item 2</li>
</ol>




<ol type="A">
  <li>Apple</li>
  <li>Banana</li>
</ol>




| Type | Output            |
| ---- | ----------------- |
| `1`  | 1, 2, 3 (default) |   
| `A`  | A, B, C           |
| `a`  | a, b, c           |
| `I`  | I, II, III        |
| `i`  | i, ii, iii        |


Strat Attribute `start="n"` to begin numbering from `n` instead of 1.
<ol start="5">
  <li>Item 5</li>
  <li>Item 6</li>
</ol>

HTML Images 

The <img> tag is used to display images on a web page.
It‚Äôs a self-closing tag ‚Äî doesn‚Äôt require a closing </img>.

<img src="image.jpg" alt="Description of image" width="300" height="200">



| Attribute | Description                          |
| --------- | ------------------------------------ |
| `src`     | Path to the image file               |
| `alt`     | Alternative text for accessibility   |  
| `width`   | Width of the image in pixels         |
| `height`  | Height of the image in pixels        |
| `title`   | Tooltip text when hovering over image |



example:
<img src="sagar.jpg" alt="Sagar Profile" width="200" height="200" title="Sagar's Photo">



Relative vs Absolute Paths
---------------------------
Relative Path (file is in your project folder):

<img src="images/logo.png" alt="Logo">


Absolute Path (from the internet):

<img src="https://example.com/logo.png" alt="Logo">

HTML Tables
Tables organize data in rows and columns.

üî∏ Basic Structure:

<table border="1" cellpadding="10" cellspacing="0">
      <tr>
        <th>Name</th>
        <th>Roll</th>
        <th>Marks</th>
      </tr>
      <tr>
        <td>Sagar</td>
        <td>101</td>
        <td>95</td>
      </tr>
    </table>




| Element | Description                |
| ------- | -------------------------- |
| `<table>` | Starts the table           |
| `<tr>`    | Table row                  |
| `<th>`    | Header cell (bold)         |
| `<td>`    | Data cell                  | 
| `<thead>` | Table head row (optional you can use <tr> insted)    |




üî∏ Extra Feature:
colspan: Merge columns

rowspan: Merge rows

<td colspan="2">Merged Cell</td>
