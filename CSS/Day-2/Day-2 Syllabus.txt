ðŸ”¹ Layers:

1. **Content** â€“ The actual text or element
2. **Padding** â€“ Space between content and border
3. **Border** â€“ The edge surrounding padding
4. **Margin** â€“ Space outside the border (between elements)

 âœ… 3. Borders

Borders wrap around elements and can be customized.

ðŸ”¹ `border-style`, `border-width`, `border-color`

```css
.box {
  border: 2px dashed red;
}
```

 ðŸ”¹ `border-radius`

Rounds the corners of the border.

```css
.card {
  border-radius: 10px;
}
```



Border Style :

dotted - Defines a dotted border
dashed - Defines a dashed border
solid - Defines a solid border
double - Defines a double border
groove - Defines a 3D grooved border. The effect depends on the border-color value
ridge - Defines a 3D ridged border. The effect depends on the border-color value
inset - Defines a 3D inset border. The effect depends on the border-color value
outset - Defines a 3D outset border. The effect depends on the border-color value
none - Defines no border
hidden - Defines a hidden border

 âœ… 4. Backgrounds

CSS allows you to add color or images behind elements.

ðŸ”¹ `background-color`

Fills the background with a solid color.

```css
body {
  background-color: #f8f9fa;
}
```

ðŸ”¹ `background-image`

Applies an image as the background.

```css
.hero {
  background-image: url('banner.jpg');
}
```

 ðŸ”¹ `background-size`

Controls the scaling of the background image.

```css
.hero {
  background-size: cover;
}
```

ðŸ”¹ `background-repeat`

Prevents the image from repeating.

```css
.hero {
  background-repeat: no-repeat;
}
```

ðŸ”¹ `background-position`

Positions the image (center, top, bottom, etc.).

```css
.hero {
  background-position: center;
}
```
